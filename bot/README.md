# bot

`bot/` は **Discord連携用Bot** を実装するディレクトリです。  
運営フローを補助する通知・確認導線を提供し、Webアプリとの接続点を担います。

## 役割

- 定例・締切・承認のリマインド通知
- 必要情報の投稿支援（テンプレート文面の補助）
- APIと連携したステータス通知
- 運用チャンネル向けの軽量オペレーション補助

## 想定技術

- Discord Bot SDK（候補: `discord.js` など）
- API連携用HTTPクライアント
- ジョブ実行基盤（スケジュール通知用）

## 推奨ディレクトリ構成（着手時の目安）

```text
bot/
├── src/
│   ├── commands/      # コマンドハンドラ
│   ├── jobs/          # 定期実行ジョブ
│   ├── services/      # API連携・業務ロジック
│   ├── templates/     # 投稿テンプレート
│   └── config/        # 設定・環境変数
├── tests/             # テスト
└── scripts/           # 運用補助スクリプト
```

## 実装方針

- Botは通知/補助に専念し、権限判定や正データ管理はAPI側で行う
- 投稿フォーマットをテンプレート化し、運用の再現性を高める
- 失敗時は再試行方針とログ出力を必須にする
